Manual EDXPKG-EDXLINUX

Este manual fornece instruções para instalar e configurar o Slim no ambiente EDXPKG-EDXLINUX.

1. Instalar o Slim

O Slim é um gerenciador de display leve. Siga estes passos para instalá-lo:

  • Abra seu terminal.
  • Instale o Slim usando o seguinte comando:
sudo apt install slim

Se o Slim já estiver instalado, você não precisa reinstalá-lo.

2. Modificar a Configuração do Slim

Para personalizar o Slim, siga estes passos:

  1. Descompacte o arquivo do Slim:

    Assumindo que você tem um arquivo slim.tar.gz, descompacte-o usando o seguinte comando:

    sudo tar -xzf slim.tar.gz
  2. Entre no diretório do Slim: 
    cd slim
  3. Entre no diretório EDXPKG: 
    cd EDXPKG
  4. Entre no diretório usr: 
    cd usr
  5. Entre no diretório share: 
    cd share
  6. Copie o arquivo slim.tar.gz para /usr/share/: 
    sudo cp -r slim.tar.gz /usr/share/
  7. Descompacte o arquivo em /usr/share/: 
    cd /usr/share/
    sudo tar -vzxf slim.tar.gz
  8. Remova o arquivo slim.tar.gz: 
    sudo rm slim.tar.gz

Este manual deve ajudá-lo a começar com o Slim no EDXPKG-EDXLINUX. Se tiver mais alguma dúvida sobre a configuração do Slim, sinta-se à vontade para perguntar!
