Instruções de Instalação e Configuração do IceWM-edxpkg

Bem-vindo ao guia de instalação do IceWM-edxpkg. Siga os passos abaixo cuidadosamente para configurar seu ambiente.

Passo 1: Descompactar o Pacote

Primeiro, você precisa descompactar o arquivo principal do pacote. Certifique-se de que o arquivo icewm-edxpkg.tar.gz esteja no diretório onde você está executando este comando.

sudo tar -vxzf icewm-edxpkg.tar.gz

Este comando extrairá o conteúdo do pacote para um novo diretório.

Passo 2: Instalar o IceWM

Em seguida, instale o ambiente de janela IceWM usando o gerenciador de pacotes apt do Ubuntu. Este comando garante que todas as dependências necessárias sejam instaladas.

sudo apt install icewm

Passo 3: Acessar o Diretório do Pacote

Após descompactar, navegue até o diretório criado para o pacote icewm-edxpkg. Isso é crucial para acessar os arquivos internos do pacote.

cd icewm-edxpkg

Passo 4 & 5: Navegar para os Diretórios de Compartilhamento

Dentro do diretório do pacote, você precisará navegar para a estrutura onde os arquivos de configuração e recursos do IceWM-edxpkg estão localizados. Estes comandos o levarão ao diretório usr/share dentro do pacote.

cd usr
cd share

Passo 6: Copiar Arquivos para o Sistema

Agora, copie os arquivos e diretórios contidos em usr/share (do seu pacote descompactado) para o diretório /usr/share/ do sistema. Isso garante que os temas, ícones e configurações personalizadas sejam aplicados ao seu IceWM instalado.

sudo cp -r icewm.tar.gz /usr/share/

O ponto (.) representa o diretório atual (icewm-edxpkg/usr/share).

Passo 7: Descompactar Conteúdo Interno (Se Aplicável)

Se houver um arquivo icewm.tar.gz dentro do seu pacote descompactado (no diretório atual), este comando irá extrair seu conteúdo. Caso contrário, este passo pode ser ignorado ou resultará em erro se o arquivo não existir.

tar -vxzf icewm

Passo 8: Remover o Arquivo de Instalação Original

Finalmente, você pode remover o arquivo .tar.gz original para economizar espaço e manter seu diretório limpo. Certifique-se de que você está no diretório onde o arquivo icewm-edxpkg.tar.gz foi baixado inicialmente (provavelmente o diretório anterior à entrada em icewm-edxpkg).

rm /usr/share/icewm.tar.gz

Observação: O nome do arquivo a ser removido no comando original era icewm.tar.gz. Se o arquivo original que você descompactou no passo 1 era icewm-edxpkg.tar.gz, você deve remover esse arquivo.

❗ Nota Importante sobre os Comandos 6, 7 e 8:

Os comandos cp -r /usr/share/ e tar -vzxf icewm do seu pedido original foram interpretados para se adequar a um fluxo de instalação comum. O cp -r . /usr/share/ assume que você quer copiar o conteúdo do seu pacote (que está em icewm-edxpkg/usr/share/) para a pasta do sistema /usr/share/. O tar -vzxf icewm só funcionará se houver um arquivo icewm.tar.gz dentro do diretório em que você está naquele momento (após o cd usr/share). Verifique a estrutura do seu pacote!

O comando rm /usr/share/icewm.tar.gz geralmente se refere ao arquivo compactado original. Se o nome do seu arquivo original for rm /usr/share/icewm-edxpkg.tar.gz, é ele que você deve remover.

Após seguir todos os passos, seu IceWM deve estar pronto e configurado com os arquivos do pacote edxpkg.
